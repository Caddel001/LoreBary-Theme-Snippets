# Images Folder

This folder contains example images and screenshots for CSS snippets in the LoreBary-Theme-Snippets guide.

## Naming Convention

Images are organized by category:
- `avatar-*.png` - Avatar customization examples
- `bubble-*.png` - Chat bubble effect examples
- `message-*.png` - Message layout examples
- `header-*.png` - Header customization examples
- `footer-*.png` - Footer customization examples
- `input-*.png` - Input control examples
- `background-*.png` - Background effect examples

## How to Add Images

1. Upload your PNG, JPG, or WebP files to this folder
2. Use the naming convention above
3. Update the Quick Guide with image references using: `![Description](./images/filename.png)`

## Image Specifications

- **Format**: PNG or WebP (recommended for web)
- **Size**: Keep under 500KB for faster loading
- **Dimensions**: 800x600px or 1024x768px for clarity
- **Resolution**: 72-96 DPI for web
